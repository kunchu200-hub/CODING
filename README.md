A simple console-based student database system in C

(This is a great mini project for students to work on)
________________________________________________________________________________________
TASK description 

The program  ask for the number of
students and for each student, ask for the following details:
Name of the university
First name and surname
Street and city
Year of study
Store this data in a structure. Store all structures in an array.
At the end, display all collected data in a readable format.

Application should ask the user about operation, user can choose: gather, display or delete data.
In the first case data gathered from the user should be saved to a file.
In the second case data read from the file should be displayed to the user.
In the last case, the last student should be deleted from the file
